# Route Intelligence Agents

Интеллектуальная система анализа маршрутов и логистики для туристического приложения.

## Возможности

### 🗺️ Интеллектуальный поиск ближайших объектов
Агент анализирует географическое расположение объектов и находит ближайшие места интереса с учетом:
- Расстояния между объектами
- Времени в пути
- Оптимального порядка посещения

### 📊 Анализ качества маршрута
Агент оценивает составленный маршрут по критериям:
- Логичность и последовательность
- Временные затраты
- Разнообразие активностей
- Баланс расстояний

### ⭐ Анализ обратной связи
Агент обрабатывает отзывы пользователей и извлекает:
- Оценки удовлетворённости
- Проблемные точки маршрута
- Рекомендации по улучшению

### 🚗 Мультимодальная маршрутизация
Агенты для разных способов передвижения:
- Автобус — общественный транспорт
- Поезд — железнодорожное сообщение
- Самолёт — авиаперелёты
- Автомобиль — личный/арендованный транспорт
- Пешком — пешие маршруты
- Велосипед — велосипедные прогулки

## Агенты

| Агент | Описание | Функция |
|-------|---------|---------|
| **NearbyFinderAgent** | Поиск ближайших объектов | Географический анализ |
| **RouteQualityAgent** | Анализ качества маршрута | Оценка и оптимизация |
| **FeedbackAnalyzerAgent** | Анализ обратной связи | Обработка отзывов |
| **DistanceCalculatorAgent** | Расчёт расстояний | Геометрические вычисления |
| **TransportAgent** | Мультимодальная маршрутизация | Оптимальный транспорт |

### Транспортные агенты

| Агент | Описание | Данные |
|-------|---------|--------|
| **BusRouteAgent** | Маршруты автобусов | Расписание, остановки |
| **TrainRouteAgent** | Железнодорожные маршруты | Поезда, станции |
| **FlightRouteAgent** | Авиамаршруты | Аэропорты, рейсы |
| **CarRouteAgent** | Автомобильные маршруты | Дороги, время в пути |
| **WalkRouteAgent** | Пешие маршруты | Тропы, сложность |
| **BikeRouteAgent** | Велосипедные маршруты | Велодорожки, расстояния |

## Использование

```bash
cd route-intelligence
pip install -r requirements.txt
python -m uvicorn main:app --reload
```

### Найти ближайшие объекты
```bash
POST /api/route/nearby
{
  "lat": 44.95,
  "lng": 38.05,
  "radius_km": 10,
  "category": "beach"
}
```

### Проанализировать маршрут
```bash
POST /api/route/analyze
{
  "locations": [{"lat": 44.95, "lng": 38.05}, ...],
  "transport": "car"
}
```

### Рассчитать расстояния
```bash
POST /api/route/distance
{
  "from": {"lat": 44.95, "lng": 38.05},
  "to": {"lat": 45.02, "lng": 38.05},
  "transport": "walk"
}
```

### Получить маршрут
```bash
POST /api/route/plan
{
  "locations": [...],
  "preferences": {
    "transport": ["car", "walk"],
    "avoid_tolls": true,
    "scenic": true
  }
}
```

## Технологии

- **FastAPI** — высокопроизводительный веб-фреймворк
- **Ollama (qwen3.5:4b)** — локальный ИИ для анализа
- **GeoPy** — геокодирование и расчёт расстояний
- **OSMNX** — данные OpenStreetMap для маршрутов
- **NetworkX** — графовые алгоритмы оптимизации

## Преимущества

### Географический интеллект
Агенты понимают пространственные отношения и находят оптимальные маршруты с учетом реальной географии Краснодарского края.

### Мультимодальность
Единый интерфейс для работы с разными видами транспорта — от пеших прогулок до авиаперелётов.

### Оптимизация в реальном времени
Алгоритмы машинного обучения оптимизируют маршруты на основе данных о трафике, погоде и предпочтениях пользователей.

### Анализ обратной связи
ИИ автоматически обрабатывает отзывы и выявляет паттерны для улучшения сервиса.

## Пример работы

```
Вход: Пользователь хочет посетить 3 места за 1 день
      - Гора Фишт (43.9, 39.4)
      - Винодельня Гай-Кодзор (44.6, 38.0)
      - Озеро Кардывач (43.8, 39.9)
      - Предпочтение: автомобиль

Обработка:
1. DistanceCalculatorAgent рассчитывает матрицу расстояний
2. RouteQualityAgent анализирует оптимальный порядок
3. CarRouteAgent проверяет дороги и время в пути
4. NearbyFinderAgent ищет дополнительные точки

Выход:
{
  "optimized_route": ["Гора Фишт" → "Озеро Кардывач" → "Винодельня"],
  "total_distance": "245 км",
  "total_time": "6 часов",
  "transport": "автомобиль",
  "quality_score": 8.5
}
```
