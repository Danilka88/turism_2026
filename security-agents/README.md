# AI Security Agents

Автономная система безопасности на основе ИИ-агентов для защиты туристического приложения.

## Возможности

### 🛡️ Мониторинг активности пользователей
Агент анализирует паттерны поведения и выявляет подозрительную активность:
- Аномальные запросы
- Попытки взлома
- Спам и флуд

### 🔍 Валидация данных парсера
Агент проверяет качество данных от парсера:
- Удаление дубликатов
- Проверка достоверности
- Фильтрация нерелевантного контента

### ✅ Модерация контента владельцев
Агент проверяет контент перед публикацией:
- Текстовые описания
- Изображения
- Цензура и фильтрация

### 🚨 Система раннего оповещения
Агент обнаруживает угрозы и оповещает:
- SQL-инъекции
- XSS-атаки
- Подозрительные IP

## Агенты

| Агент | Описание | Функция |
|-------|---------|---------|
| **UserBehaviorAgent** | Анализ поведения пользователей | Выявление аномалий |
| **DataQualityAgent** | Контроль качества данных парсера | Валидация контента |
| **ContentModeratorAgent** | Модерация контента владельцев | Проверка публикаций |
| **ThreatDetectorAgent** | Обнаружение угроз | Защита от атак |
| **AuditLoggerAgent** | Логирование действий | Аудит и отчётность |

## Архитектура

```
security-agents/
├── agents/
│   ├── UserBehaviorAgent.py      # Поведенческий анализ
│   ├── DataQualityAgent.py       # Качество данных
│   ├── ContentModeratorAgent.py  # Модерация контента
│   ├── ThreatDetectorAgent.py    # Детекция угроз
│   └── AuditLoggerAgent.py       # Аудит логирования
├── services/
│   ├── OllamaSecurityService.py  # Ollama для анализа
│   └── AlertService.py           # Система оповещений

│   └── routes.py                 # FastAPI эндпоинты
├── models/
│   └── schemas.py               # Pydantic модели
├── prompts/
│   └── security_prompts.py       # Промты для агентов
└── main.py                      # Точка входа
```

## Использование

```bash
cd security-agents
pip install -r requirements.txt
python -m uvicorn main:app --reload
```

### Проверка пользователя
```bash
POST /api/security/check-user
{
  "user_id": "user_123",
  "action": "login",
  "ip": "192.168.1.1",
  "user_agent": "Mozilla/5.0..."
}
```

### Валидация данных парсера
```bash
POST /api/security/validate-data
{
  "content": "Описание места...",
  "images": ["url1", "url2"],
  "source": "google_maps"
}
```

### Модерация контента
```bash
POST /api/security/moderate
{
  "content": "Текст от владельца",
  "type": "description"
}
```

## Технологии

- **FastAPI** — высокопроизводительный веб-фреймворк
- **Ollama (qwen3.5:4b)** — локальный ИИ для анализа
- **PostgreSQL** — хранение логов и метрик
- **Redis** — кэширование и rate limiting
- **Prometheus** — метрики безопасности

## Преимущества

### Полная автономность
Агенты работают независимо от основного приложения, не влияя на производительность.

### Локальный ИИ
Все модели работают локально через Ollama — никаких утечек данных.

### Многоуровневая защита
- Поведенческий анализ
- Валидация контента
- Детекция угроз в реальном времени

### Аудит
Полное логирование всех действий для соответствия требованиям.

## Метрики безопасности

```
• % обнаруженных угроз
• Среднее время реакции на инцидент
• Качество модерации контента
• Точность поведенческого анализа
```

## Интеграция с основным приложением

Система работает как отдельный микросервис и взаимодействует с основным приложением через API.

```
Основное приложение  ←→  Security Agents API
         ↓                    ↓
    Парсер данных      Ollama (qwen3.5:4b)
         ↓                    ↓
         ←────── Анализ ─────→
```
