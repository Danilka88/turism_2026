# Привет, Краснодарский край

[🎬 Видео обзор платформы](https://disk.yandex.ru/i/YM6EWY1egrVt0Q)

Приложение для подбора туристических маршрутов по Краснодарскому краю с ИИ-агентами.

## Возможности

### 🗺️ Путешествие по карте
Интерактивная карта Краснодарского края с **68+ точками интереса**. Выбирайте места вручную — фильтруйте по категориям (пляжи, горы, винодельни, природа, культура, SPA, семья, города) и стройте свой маршрут. При выборе точки на карте появляется **тепловая карта расстояний** — зелёным отмечены близкие места, жёлтым и красным — более далёкие.

### 🍷 Сканер бутылки вина (Vision Agent)
Уникальная функция для любителей вина. Сфотографируйте бутылку — VisionAgent распознает вино по **базе виноделен Кубани** (Фанагория, Абрау-Дюрсо, Гай-Кодзор, Шато Тамань) и предложит:
- Информацию о вине и производителе
- Маршрут на винодельню
- Рекомендации по сочетанию с едой

### ⚡ Экстренный маршрут
Когда планы меняются — погода испортилась, объект закрыт, скучно или хочется чего-то нового. Выберите проблему из 6 карточек и получите **готовый альтернативный маршрут**.

### 💕 Подбор маршрута (ИИ-агенты)
Пройдите короткий онбординг — укажите интересы, компанию и даты. ИИ-агенты проанализируют профиль и подберут лучшие места.

### 📋 Кабинет владельца
Для владельцев туристических объектов — редактирование информации о локациях прямо в приложении.

### 🧪 Тестирование агентов
Нажмите кнопку **"Тест"** в правом верхнем углу для проверки работы всех ИИ-агентов.

## ИИ-агенты (Ollama)

| Агент | Описание | Задержка |
|-------|----------|----------|
| **OnboardingAgent** | Профилирование пользователя по интересам | 1.5-3 сек |
| **RouteMatcherAgent** | Подбор мест по интересам и компании | 1.5-3 сек |
| **VisionAgent** | Анализ фото бутылки + база виноделен Кубани | 10-15 сек |
| **EmergencyRouteAgent** | Альтернативы при проблемах (погода, закрыто и т.д.) | 1.5-3 сек |
| **DescriptionGeneratorAgent** | Генерация описаний локаций | 1.5-3 сек |
| **PersonalizationAgent** | Обучение на действиях пользователя | 1.5-3 сек |

### VisionAgent - База виноделен

| Винодельня | Регион | Сорта винограда |
|-----------|--------|-----------------|
| Фанагория | станица Натухаевская | Каберне, Мерло, Шардоне, Рислинг, Алиготе |
| Абрау-Дюрсо | Новороссийск | Пино Нуар, Шардоне, Каберне, Мерло |
| Гай-Кодзор | Геленджик | Каберне, Мерло, Совиньон |
| Шато Тамань | Тамань | Каберне, Шардоне, Мерло |

### Режимы работы

- **Ollama (рекомендуется)** — использует локальную модель `qwen3.5:4b` (показывает зелёный индикатор)
- **Demo** — fallback режим с демо-данными (работает без Ollama, жёлтый индикатор)

## Запуск

### Быстрый старт (Demo-режим)

```bash
npm install
npm run dev
```

Приложение будет доступно по адресу: http://localhost:3000

> Без Ollama приложение работает в демо-режиме с заранее подготовленными данными.

### Запуск с Ollama (полный функционал)

```bash
# 1. Установите Ollama
brew install ollama  # macOS
# или скачайте с https://ollama.com

# 2. Запустите Ollama сервер (в отдельном терминале)
ollama serve

# 3. Скачайте модель (в отдельном терминале)
ollama pull qwen3.5:4b

# 4. Запустите приложение
npm run dev
```

При успешном подключении увидите индикатор **"Ollama"** (зелёный).

## Парсер данных (parser/)

Автономный Python-парсер для сбора актуальной информации о туристических местах Краснодарского края.

**Возможности:**
- Поиск в Google/Yandex с обходом анти-бота
- Парсинг изображений из Google Images
- Поиск видео на Rutube
- Агрегирование результатов из разных источников

**Стек:** FastAPI + Selenium + undetected-chromedriver

**Статус:** Готов к интеграции

```bash
cd parser
pip install -r requirements.txt
python -m uvicorn app.main:app --reload
```

### Как это работает

```
Запрос: "Фанагория винодельня"
         ↓
┌─────────────────────────────────────┐
│  SearchService (Selenium Chrome)     │
├─────────────┬─────────────┬─────────┤
│ TextSearch  │ImageSearch  │Rutube   │
│ Parser      │Parser       │Parser   │
└─────────────┴─────────────┴─────────┘
         ↓
Агрегированные результаты с фото и видео
```

### Планы интеграции

- Автоматическое обновление данных о локациях
- Сбор актуальных отзывов и фото
- Добавление видео-контента в карточки

## AI Security Agents

Автономная система безопасности на основе ИИ-агентов для защиты туристического приложения.

### Возможности

**🛡️ Мониторинг активности пользователей**
Агент анализирует паттерны поведения и выявляет подозрительную активность: аномальные запросы, попытки взлома, спам и флуд.

**🔍 Валидация данных парсера**
Агент проверяет качество данных от парсера: удаление дубликатов, проверка достоверности, фильтрация нерелевантного контента.

**✅ Модерация контента владельцев**
Агент проверяет контент перед публикацией: текстовые описания, изображения, цензура и фильтрация.

**🚨 Система раннего оповещения**
Агент обнаруживает угрозы и оповещает: SQL-инъекции, XSS-атаки, подозрительные IP.

### Агенты безопасности

| Агент | Описание | Функция |
|-------|---------|---------|
| **UserBehaviorAgent** | Анализ поведения пользователей | Выявление аномалий |
| **DataQualityAgent** | Контроль качества данных парсера | Валидация контента |
| **ContentModeratorAgent** | Модерация контента владельцев | Проверка публикаций |
| **ThreatDetectorAgent** | Обнаружение угроз | Защита от атак |
| **AuditLoggerAgent** | Логирование действий | Аудит и отчётность |

### Запуск

```bash
cd security-agents
pip install -r requirements.txt
python -m uvicorn main:app --reload
```

### Архитектура безопасности

```
Основное приложение  ←→  Security Agents API
         ↓                    ↓
    Парсер данных      Ollama (qwen3.5:4b)
         ↓                    ↓
         ←────── Защита ──────→
```

### Преимущества

- **Полная автономность** — агенты работают независимо, не влияя на производительность
- **Локальный ИИ** — все модели через Ollama, никаких утечек данных
- **Многоуровневая защита** — поведенческий анализ, валидация контента, детекция угроз
- **Аудит** — полное логирование для соответствия требованиям

## Route Intelligence Agents

Интеллектуальная система анализа маршрутов и логистики для оптимального планирования путешествий.

### Возможности

**🗺️ Интеллектуальный поиск ближайших объектов**
Агент анализирует географическое расположение и находит ближайшие места интереса с учетом расстояния, времени в пути и оптимального порядка посещения.

**📊 Анализ качества маршрута**
Агент оценивает составленный маршрут по критериям: логичность, временные затраты, разнообразие активностей и баланс расстояний.

**⭐ Анализ обратной связи**
ИИ обрабатывает отзывы пользователей, извлекает оценки удовлетворённости, выявляет проблемные точки и даёт рекомендации по улучшению.

**🚗 Мультимодальная маршрутизация**
Агенты для разных способов передвижения:
- 🚗 Автомобиль — личный/арендованный транспорт
- 🚌 Автобус — общественный транспорт
- 🚂 Поезд — железнодорожное сообщение
- ✈️ Самолёт — авиаперелёты
- 🚶 Пешком — пешие маршруты
- 🚴 Велосипед — велосипедные прогулки

### Агенты маршрутизации

| Агент | Описание | Функция |
|-------|---------|---------|
| **NearbyFinderAgent** | Поиск ближайших объектов | Географический анализ |
| **RouteQualityAgent** | Анализ качества маршрута | Оценка и оптимизация |
| **FeedbackAnalyzerAgent** | Анализ обратной связи | Обработка отзывов |
| **DistanceCalculatorAgent** | Расчёт расстояний | Геометрические вычисления |

### Запуск

```bash
cd route-intelligence
pip install -r requirements.txt
python -m uvicorn main:app --reload
```

### Как это работает

```
Вход: Пользователь хочет посетить 3 места за 1 день
      - Гора Фишт (43.9, 39.4)
      - Винодельня Гай-Кодзор (44.6, 38.0)
      - Озеро Кардывач (43.8, 39.9)
      - Предпочтение: автомобиль

Обработка:
1. DistanceCalculatorAgent рассчитывает матрицу расстояний
2. RouteQualityAgent анализирует оптимальный порядок
3. NearbyFinderAgent ищет дополнительные точки

Выход:
{
  "optimized_route": ["Гора Фишт" → "Озеро Кардывач" → "Винодельня"],
  "total_distance": "245 км",
  "total_time": "6 часов",
  "quality_score": 8.5
}
```

### Преимущества

- **Географический интеллект** — понимание пространственных отношений и оптимизация маршрутов
- **Мультимодальность** — единый интерфейс для всех видов транспорта
- **Оптимизация в реальном времени** — алгоритмы машинного обучения для лучших маршрутов
- **Анализ отзывов** — автоматическая обработка обратной связи для улучшения сервиса
