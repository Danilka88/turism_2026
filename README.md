# Привет, Краснодарский край

Приложение для подбора туристических маршрутов по Краснодарскому краю.

## Возможности

### 🗺️ Путешествие по карте
Интерактивная карта Краснодарского края с **68+ точками интереса**. Выбирайте места вручную — фильтруйте по категориям (пляжи, горы, винодельни, природа, культура, SPA, семья, города) и стройте свой маршрут. При выборе точки на карте появляется **тепловая карта расстояний** — зелёным отмечены близкие места, жёлтым и красным — более далёкие.

### 🍷 Сканер бутылки вина
Уникальная функция для любителей вина. Сфотографируйте бутылку — приложение распознает винодельню и автоматически предложит:
- Маршрут на эту винодельню
- Близлежащие интересные места
- Рекомендации что взять с собой

### ⚡ Экстренный маршрут
Когда планы меняются — погода испортилась, объект закрыт, скучно или хочется чего-то нового. Выберите проблему из 6 карточек и получите **готовый альтернативный маршрут за секунды**. Идеально для импровизированных приключений.

### 💕 Подбор маршрута
Пройдите короткий онбординг — укажите интересы, компанию и даты. Приложение покажет карточки локаций, где вы выбираете:
- Добавить в маршрут
- Пропустить
После выбора система формирует финальный маршрут с возможностью добавить **еду и активности** для каждого места.

### 📋 Кабинет владельца
Для владельцев туристических объектов — редактирование информации о локациях прямо в приложении.

## ИИ-агенты (Ollama)

Приложение использует **модульную систему ИИ-агентов** на базе Ollama:

| Агент | Описание |
|-------|----------|
| **OnboardingAgent** | Профилирование пользователя по интересам |
| **RouteMatcherAgent** | Подбор мест по интересам и компании |
| **WineScannerAgent** | Распознавание вина и рекомендации |
| **EmergencyRouteAgent** | Альтернативы при проблемах в пути |
| **DescriptionGeneratorAgent** | Генерация описаний локаций |
| **PersonalizationAgent** | Обучение на действиях пользователя |

### Режимы работы

- **Ollama (рекомендуется)** — использует локальную модель `qwen3.5:4b`
- **Demo** — fallback режим с демо-данными (работает без Ollama)

## Запуск

### Быстрый старт (Demo-режим)

```bash
npm install
npm run dev
```

Приложение будет доступно по адресу: http://localhost:3000

> Без Ollama приложение работает в демо-режиме с заранее подготовленными данными.

### Запуск с Ollama (полный функционал)

1. **Установите Ollama**: https://ollama.com

2. **Скачайте модель**:
```bash
ollama pull qwen3.5:4b
```

3. **Запустите Ollama сервер** (в отдельном терминале):
```bash
ollama serve
```

4. **Запустите приложение**:
```bash
npm run dev
```

При успешном подключении на экране загрузки появится индикатор "Ollama qwen2.5:4b".

## Технологии

- **React 19** + **TypeScript** — современный фреймворк
- **Vite** — быстрая сборка
- **Tailwind CSS 3** — стилизация
- **Leaflet** + **React-Leaflet** — интерактивные карты
- **Framer Motion** — плавные анимации
- **Lucide React** — иконки
- **Ollama** — локальные ИИ-агенты
